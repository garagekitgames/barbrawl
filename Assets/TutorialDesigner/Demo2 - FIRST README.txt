Demo2 Import Instructions

This Demo needs TextMeshPro loaded and activated within TutorialDesigner. Follow these steps to do so:

1. If you have Unity 2017 or below, Install TextMeshPro from the Unity Asset Store. This doesn't apply for Unity 2018
2. Load TextMeshPro Module for TutorialDesigner. Window -> TutorialDesigner -> Load TextMeshPro Module
3. Doubleclick Demo2 (unitypackage) in the Projects window and import it
4. Now you can open the Demo Scene: Demo2 -> Scenes -> Demo

Hints: This includes some Unity Standard Assets that will be in the root folder of your project
